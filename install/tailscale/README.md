# Installing Tailscale

The tailscale documentation states the way to install is to run:

```sh
  curl -fsSL https://tailscale.com/install.sh | sh
```

The following files are stored as .assets/:

| File                          | Description                                                                                                                 |
|-------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| install.sh                    | The install.sh script from the tailscript URL (as it was on 24th March 2026).                                               |
| install-distilled.README      | The distilled version which is meant to be read (not executed) so you can understand what the install.sh is actually doing. |
| tailscale-archive-keyring.gpg | A copy of the (binary) public key that the install script will copy to local disk                                           |
